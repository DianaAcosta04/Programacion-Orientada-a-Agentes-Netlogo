Para poder ejecutar los modelos de los talleres 1,2 y 4, debes tener instalado NetLogo o, si estás en la versión online, puedes descargar el archivo desde los tres puntos a la derecha del archivo del taller, y subes el archivo en la versión online, si estás en la versión de escritorio (teniendo instalado el programa de NetLogo) puedes descargar directamente el modelo y se abrirá automáticamente.
